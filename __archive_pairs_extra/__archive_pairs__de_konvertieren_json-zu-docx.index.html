Ausgaben

Schnellstart

  1. Dateien hinzufügen: per Drag & Drop in die Box oder auf Dateien hinzufügen klicken.
  2. Ausgabe wählen: ein Format unter Konvertieren in wählen (für JPEG/WebP Qualität setzen).
  3. Konvertieren: Konvertieren drücken. Die Zeile zeigt Badge, Fortschritt und Status.
  4. Download: wenn der Status Bereit ist, Dateiname klicken, um das Ergebnis herunterzuladen. Mit Alle speichern alle Ausgaben auf einmal speichern.
  5. Entfernen: auf das × einer Datei klicken, um sie aus der Liste zu entfernen.

Tipps & Tricks

  • Parallele Konvertierungen: einstellen, wie viele gleichzeitig laufen.
  • Speicherbudget: auf Manuell stellen, wenn Limits erreicht werden.
  • Lange Namen: Namen werden auf der Karte gekürzt; zum Anzeigen vollständiger Namen darüberfahren.
  • Privatsphäre: alles läuft im Browser — keine Uploads.
