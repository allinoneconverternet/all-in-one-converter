Resultados

Início rápido

  1. Adicionar arquivos: arraste e solte na caixa ou clique em Adicionar arquivos.
  2. Escolher saída: selecione um formato em Converter para (defina a Qualidade para JPEG/WebP).
  3. Converter: clique em Converter. A linha mostra um selo, progresso e status.
  4. Baixar: quando o status estiver Pronto, clique no nome do arquivo para baixar o resultado. Use Salvar tudo para salvar todas as saídas de uma vez.
  5. Remover: clique no × em um arquivo para removê‑lo da lista.

Dicas

  • Conversões paralelas: ajuste quantas rodam ao mesmo tempo.
  • Orçamento de memória: mude para Manual se atingir limites.
  • Nomes longos: os nomes são encurtados no cartão; passe o mouse para ver o nome completo.
  • Privacidade: tudo roda no navegador — sem upload.
