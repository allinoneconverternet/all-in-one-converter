Risultati

Guida rapida

  1. Aggiungi file: trascina nella casella o fai clic su Aggiungi file.
  2. Scegli uscita: seleziona un formato in Converti in (imposta la Qualità per JPEG/WebP).
  3. Convertii: premi Convertii. La riga mostra badge, progresso e stato.
  4. Scarica: quando lo stato è Pronto, fai clic sul nome del file per scaricare il risultato. Usa Salva tutto per salvare tutte le uscite in una volta.
  5. Rimuovi: fai clic su × in un file per rimuoverlo dall’elenco.

Suggerimenti

  • Conversioni in parallelo: regola quante eseguire contemporaneamente.
  • Budget di memoria: passa a Manualee se raggiungi i limiti.
  • Nomi lunghi: i nomi sono abbreviati sulla scheda; passa sopra per vedere il nome completo.
  • Privacy: tutto gira nel browser — nessun upload.
